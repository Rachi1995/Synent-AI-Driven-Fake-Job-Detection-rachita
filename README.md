# Fake Job Detection System

# Project Overview

The Fake Job Detection System is an AI-powered web application designed to identify fraudulent job postings using Machine Learning and Natural Language Processing (NLP).
With the rise of online recruitment platforms, fake job advertisements have become a major issue for students, freshers, and job seekers. This project helps users analyze suspicious job descriptions and determine whether a posting is genuine or fraudulent.
The application provides a simple and interactive interface where users can paste a job description and instantly receive fraud analysis results.

# Objective

The main objective of this project is to:
- Detect fake or scam job postings
- Help users avoid online recruitment fraud
- Analyze suspicious hiring language
- Build a complete End-to-End Data Science project with deployment

# 🌍 Real-World Problem

Many fake job postings contain:
- Unrealistic salary promises
- Instant hiring offers
- Work-from-home scams
- Registration fee requests
- Suspicious hiring language

This system helps identify such patterns and alert users before they apply.

# Key Features

AI-powered fake job detection  
Real-time prediction system  
Modern and interactive UI  
Confidence score display  
Scam keyword analysis  
Instant fraud alerts  
User-friendly web interface  
Fast response system  

# What the System Analyzes

The system checks:
- Job title
- Company profile
- Job description
- Hiring language
- Suspicious keywords
- Scam-related phrases

It identifies common fraud indicators found in fake recruitment advertisements.
